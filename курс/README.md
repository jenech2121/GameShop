# GameShop 
```
GameShop/курс/
├── Kursovaya/          # Frontend (Vue 3)
└── Client-server/      # Backend (Spring Boot)
```

### Запуск Frontend

```bash
cd Kursovaya
npm install
npm run dev
```

Frontend будет доступен на: http://localhost:5173

### Запуск Backend

```bash
cd Client-server
mvn clean install
mvn spring-boot:run
```
